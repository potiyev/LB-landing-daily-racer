# Loop Break — Daily Driver · Landing Page

Static marketing site for the indie racing game **Loop Break — Daily Driver** (iOS / Android, launching soon).

- `index.html` — landing page
- `privacy.html` — direct-linkable privacy policy (for App Store / Play Store submission)
- `support.html` — FAQ + contact

Pure HTML/CSS/JS. No build step. Drop into GitHub Pages and ship.

To preview locally, open `index.html` directly in a browser, or serve the folder with any static server (e.g. `python3 -m http.server`).
